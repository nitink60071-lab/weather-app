# 🌦️ Weather App by Nitin

A modern and responsive Weather App built with **React.js** and **Material UI** that provides real-time weather information for any city using the OpenWeather API.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature
- 💧 Humidity information
- 🌡️ Minimum & Maximum temperature
- ☁️ Weather description
- 🌤️ Dynamic weather images based on temperature and humidity
- 🌦️ Weather icons for different weather conditions
- 📱 Responsive and user-friendly UI

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- Material UI (MUI)
- HTML5
- CSS3
- OpenWeather API
